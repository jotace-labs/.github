## Technology Stack

### Core Technologies
- **Go (Golang)**: Primary language for building microservices
- **Kubernetes**: Container orchestration platform
- **ArgoCD**: GitOps continuous delivery tool for Kubernetes
- **k8s-argo-apps**: Deployment configuration and management

### Architecture
Projects in this organization follow a microservices architecture pattern, where:
- Each service is independently deployable
- Services are containerized using Docker
- Kubernetes manifests define service deployments
- ArgoCD handles GitOps-based deployments
- Infrastructure follows declarative configuration principles
